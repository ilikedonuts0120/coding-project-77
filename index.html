1. Download from the website 2. Activate application 3. Go to 'File' and turn autosave on 4. Install the extensions 'Live Server' and 'Beautify' 5. Create a folder 6. Open the folder 7. Create a file within the folder titled 'index.html'

The console screen is used to check error messages while coding. To open the console screen, right click on the website and select 'Inspect'. Then click on the tab labeled 'Console'. To print something in the console, write 'console.log=' and then whatever you want to say.
